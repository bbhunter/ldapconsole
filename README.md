# ldapconsole

The ldapconsole script allows you to perform custom LDAP requests to a Windows domain.

![](./.github/example.png)

## Features

 - [x] Authenticate with password
 - [x] Authenticate with LM:NT hashes
 - [x] Authenticate with kerberos ticket
